# Quantum X Builder - Quick Start Guide

## Getting Started in 30 Seconds

1. **Open the application**
   ```bash
   # Start a simple web server
   python3 -m http.server 8080
   
   # Open in your browser
   # Navigate to: http://localhost:8080/demo.html
   ```

2. **Start coding!**
   - The editor comes pre-loaded with example code
   - Start typing to see AI suggestions
   - Try typing keywords like `func`, `class`, or `async`

## Tips & Tricks

### Smart Suggestions
- Type partial keywords to trigger suggestions
- JavaScript: `func`, `class`, `async`, `arrow`, `for`, `try`
- Python: `func`, `class`, `for`, `if`, `try`, `with`
- TypeScript: `interface`, `type`, `func`, `class`

### Quick Actions
- **Explain Code**: Select code or use on entire file
- **Optimize**: Get performance and quality tips
- **Refactor**: Get code improvement suggestions
- **Debug**: Find syntax and logic issues
- **Add Docs**: Generate documentation templates
- **Generate Tests**: Create unit test templates

### Keyboard Shortcuts
- `Tab` key: Indent code (4 spaces)
- `Enter`: New line
- Text selection: Select code to explain or work with
