# Vizual-X Privacy Policy

**Vizual-X** is an autonomous control-plane system designed for governed, auditable AI-assisted operations.

## Data Handling

Vizual-X does not collect, store, or retain personal user data from ChatGPT conversations. All requests received from ChatGPT Actions are treated as transient, machine-generated requests and are processed solely for the purpose of executing authorized system operations.

## Authentication & Authorization

All authentication, authorization, and policy enforcement occur server-side within Vizual-X infrastructure. ChatGPT does not receive or store credentials, tokens, or secrets.

## Data Access

Vizual-X may access system metadata, repository information, and operational state strictly within the permissions granted to its backend services (e.g., GitHub App installations). No personal GitHub user data is accessed unless explicitly authorized through governed system policy.

## Logging & Auditing

Operational requests may be logged for security, auditing, and compliance purposes. Logs are restricted to system-level metadata and do not include ChatGPT conversation content beyond what is required to execute a request.

## Third-Party Services

Vizual-X may interact with third-party platforms (such as GitHub or cloud infrastructure providers) solely to fulfill authorized operations. All such interactions are governed by internal policy controls and least-privilege access.
