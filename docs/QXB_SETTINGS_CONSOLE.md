# QXB Settings Console

The Settings Console is the control surface for connectors, autonomy modes, budgets, trust zones, and audit.

## Sections

- Connections (OAuth/API Keys/Service Accounts)
- Connector Catalog (UUIDs, capabilities, scopes, policies)
- Autonomy (AUTO/HYBRID/MANUAL, per-level toggles)
- Budgets (hard/soft limits)
- Trust Zones (Z0/Z1/Z2)
- Audit Ledger
- Webhooks + Schedules
